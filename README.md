Proyecto final El Arte de Programar 2025

Descripción del proyecto:
El código lee un archivo csv que contiene información de varias figuras e imprime las medidas de las figuras y sus áreas, las cuales calcula.

El proyecto hace uso de pandas, debe instalarse antes de usar el código con el comando:
pip install pandas

para ejecutar escribe
python3 -m venv /Users/miusuario/micarpeta/miapp
python3 figurasPrincipal.py
